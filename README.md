# Bond YTM Calculator (MOEX Formula 12)

Полноценный Python-пакет для точного расчёта доходности к погашению (YTM) по методологии Московской Биржи (Формула 12).

## Особенности

- **Точная формула**: Реализация формулы 12 из методик МОЭКС (дисконтирование по сложной ставке).
- **Автоматизация**: Автоматическое получение данных из MOEX ISS (рыночные цены, НКД, графики купонов и амортизаций).
- **Сложные случаи**: 
    - Обработка амортизации (изменение номинала).
    - Расчёт для бессрочных облигаций (Perpetual).
    - Автоматический расчёт неизвестных будущих купонов по методу pro-rata.
    - Переключение на простую доходность в последний год/период.
- **Production-ready**: 
    - Полная типизация (Pydantic).
    - Логирование через `structlog`.
    - Ретраи и кэширование запросов к ISS.
    - Модульная архитектура.

## Быстрый запуск (Интерактивный режим)

Если вы хотите быстро рассчитать доходность конкретной бумаги, запустите скрипт `main.py`:

```bash
python main.py
```
Программа попросит ввести ISIN или SECID (например, `SU26207RMFS9`) и выведет подробную таблицу платежей и итоговую доходность.

## Использование как библиотеки (API)

Для интеграции в свои проекты используйте класс `BondYieldCalculator`:

```python
from bond_ytm import BondYieldCalculator
from datetime import date

# Создаем калькулятор
calc = BondYieldCalculator()

# Рассчитываем доходность для ОФЗ 26207
# Библиотека сама найдет активный борд (например, TQOB) и актуальную цену
result = calc.calculate(secid="SU26207RMFS9")

if result.ytm_percent is not None:
    print(f"SECID: {result.secid}")
    print(f"Доходность: {result.ytm_percent}% годовых")
    print(f"Грязная цена: {result.dirty_price}")
    print(f"НКД: {result.accrued_interest}")
    print("\nГрафик будущих платежей:")
    for flow in result.cashflows[:5]:
        type_str = " + ".join([t for t, cond in [("Купон", flow.is_coupon), ("Амортизация", flow.is_amortization)] if cond])
        print(f"{flow.date}: {flow.amount:8.2f} ({type_str:15}, Ост.номинал: {flow.remaining_face_value})")
else:
    print(f"Ошибка: {result.error_reason}")
```

## Структура пакета

- `config.py`: Константы, YearBasis, настройки ретраев.
- `models.py`: Pydantic-модели для данных и результатов.
- `fetcher.py`: Клиент для MOEX ISS API.
- `cashflows.py`: Логика построения денежных потоков.
- `solver.py`: Математический движок (scipy.optimize.brentq).
- `calculator.py`: Главный фасад (Public API).

## Методология

Расчёт ведется согласно формуле (12) из внутренних методик МОЭКС:
$P + A = \sum \frac{C_i}{(1 + Y/100)^{t_i / YearBasis}} + \dots$

Где $P$ — чистая цена, $A$ — НКД, $C_i$ — выплаты.
